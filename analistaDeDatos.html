Ingeniero de Software Senior

Tech Solutions Inc. · Remote

Descripción del puesto

Tech Solutions Inc. Está buscando un ingeniero de Software Senior altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato tendrá una sólida formación en el desarrollo de software, con experiencia en el diseño de, desarrollo e implementación de soluciones de software escalables y de alto rendimiento. Como ingeniero de software senior, usted será responsable de liderar proyectos de desarrollo, mentorizar a ingenieros junior y colaborar con equipos multifuncionales para entregar productos softwarede alta calidad.

Responsabilidades

  • Diseñar, desarrollar y mantener aplicaciones web utilizando tecnologías modernas

  • Colaborar con equipos de producto para definir y entregar nuevas caracteristicas

  • Escribir código limpio, eficiente y bien documentado

  • Realizar revisiones de código y proporcionar retroalimentación constructiva a los miembros del equipo

Requisitos

  • Licenciatura en informática o campo relacionado

  • Mínimo de 5 años de experiencia en desarrollo de software

  • Experiencia con frameworcks de JavaScript (por ejemplo React, Angular)

    Familiaridad con metodologías ágiles y herramientas de control de versiones

Acerca de la empresa

Tech Solution Inc. Es una empresa de tecnología innovadora que se centra en la creación de software de vanguardia para diversas industrias. Estamos comprometidos con el fomento de un entorno de trabajo colaborativo e inclusivo donde cada empleado pueda prosperar y crecer profesionalmente.
